
# 调查数据集（TMDb电影数据）

## 简介

数据集選擇“电影数据库”（TMDb，The Movie Database），該包含 1 万条电影信息，其中包括用户评分和票房。“演职人员 (cast)”、“电影类别 (genres)”等数据列。


## 步骤

+ 数据整理
+ 探索性数据分析
+ 结论


## 探索内容：

    *票房收入最高值及該電影信息
    *影響票房收入的相關因素
    *票房收入和知名度
    *票房收入和電影時長
    *票房收入和電影評分
    *票房收入和電影預算
    *電影的發展特點

## 問題： 

    *近年來電影票房收入呈現什麽趨勢？
    *票房收入最高的電影是哪一部？其票房值為多少？
    *電影呈現什麽樣的發展趨勢呢？
     --票房收入，知名度，電影時長，電影評分
    *決定票房收入的相關因素有什麽呢？
